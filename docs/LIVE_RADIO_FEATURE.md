# Live Radio Feature - User Guide

## What is Live Radio?

The Live Radio feature provides a 24/7 continuous stream of naats where all users listen to the same naat at the same time, creating a shared spiritual experience.

## Technical Details

### How It Works

The live radio uses time-based synchronization:

1. A server tracks which naat is currently "playing" and when it started
2. When you join, the app calculates where in the naat you should be
3. Everyone plays the same naat at the same position = synchronized!

### Why No Seek Controls?

To maintain synchronization, seek controls are disabled. This ensures everyone hears the same thing at the same time.

### Data Usage

Live radio uses the same amount of data as regular playback - it's not actually streaming, just synchronized playback of regular audio files.
