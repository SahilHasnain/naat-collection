# Android CI

## Docker Setup (Recommended)

### Prerequisites
- Docker and Docker Compose installed
- GitHub Personal Access Token with `repo` scope
- Android signing keys in `/home/Sahilhasnain/android-secrets/`

### Quick Start

1. **Copy environment file:**
   ```bash
   cd infra/android-ci
   cp .env.example .env
   ```

2. **Edit `.env` file:**
   ```bash
   GITHUB_TOKEN=ghp_your_token_here
   GITHUB_REPO=your-username/naat-collection
   RUNNER_NAME=android-docker-runner
   ```

3. **Build and start the runner:**
   ```bash
   docker-compose up -d
   ```

4. **Check logs:**
   ```bash
   docker-compose logs -f
   ```

5. **Update workflow to use new runner label:**
   ```yaml
   runs-on: [self-hosted, linux, x64, android-docker]
   ```

### What's Included

- Ubuntu 22.04 base
- Node.js 20
- Java 17 JDK
- Ruby with Bundler
- Android SDK with Platform Tools, Android 34, Build Tools 34.0.0, NDK 27.1.12297006
- GitHub Actions Runner 2.334.0

### Troubleshooting

**View logs:**
```bash
docker-compose logs -f github-runner
```

**Rebuild:**
```bash
docker-compose down
docker-compose build --no-cache
docker-compose up -d
```

**Access shell:**
```bash
docker exec -it github-actions-android-runner bash
```
